Project #4 Exquisite Corpse -9/28/2021

Design Process

Exquisite corpse is a surrealist technique where players take turns to draw a portion of a figure while not being able to see what has already been drawn by other players. For this assignment, I have exchanged the body parts with my partners:Beatriz Ribeiro and KT Son, and assembled into one figure showing above.

Reflection

By introducing customized functions,I realized the importance of the structure for coding. Before, my coding was process oriented, which is following my drawing process, coded step by step, layer by layer from the bottom to the front. With customized functions and function.this, I was able to start structure coding to object oriented, by making them out of objects that interact with one another. However, I think that I still have a long journey to simplify my code and program in a much more efficient way.
